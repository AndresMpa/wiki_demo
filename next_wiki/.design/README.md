## Sobre el diseño

Se optó por bosquedar en Figma lo que se sería desarrollar en terminos de frontend, para este proposito se tomó como insumo el siguiente material:

- [Guía de diseño](https://bootcamp.uxdesign.cc/understanding-the-basics-of-ui-ux-design-a-beginners-guide-11752310317a)

Una de las primeras problemas que se encontró fue el hecho de que la pagina actual es muy "aburrida", redunda mucho y sus contenidos son algo complejos de seguir para las generaciones actuales, tomando la guía complementaria tiene mucho diseño "F"
